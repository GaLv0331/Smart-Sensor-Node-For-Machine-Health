# STM32 Firmware

## Responsibilities
- ADC sampling for ACS712
- SPI communication with ADXL345
- Signal processing (RMS, FFT)
- Autoencoder inference
- Health score calculation
- UART/SPI communication with ESP32

## Sampling Details
- ADC Sampling Rate: 4 kHz
- RMS Window: 4 ms

## ML Inference
- Fixed-point optimized
- Reconstruction error based anomaly detection
